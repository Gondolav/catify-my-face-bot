# catify-my-face-bot

A Telegram bot that replaces human faces with cats.

## Installation

First, install the required modules:

```
pip install -r requirements.txt
```

Then, create a file `.keys` with the following content:

```
token=<your-telegram-bot-token>
```

## Usage

Run the bot with:

```
python bot.py
```
